Wokenv

The Perfect Blend for WordPress Development

Docker-based WordPress development environment with Node.js, Docker, and @wordpress/env pre-configured.
No Node.js installation needed. Just run and develop.

Get Started in Seconds

curl -fsSL https://raw.githubusercontent.com/wokenv/wokenv/main/install.sh | bash

Then run wokenv init in your project directory
